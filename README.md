# Night Tracker

A calm, dark-mode iOS app for parents to log and understand their baby's night-time wakeups.

Built with SwiftUI, targeting iOS 17+. No third-party dependencies — just Apple frameworks and XcodeGen for project generation.

## Features

### Profiles
Switch between multiple babies (or parents) from a chip picker at the top of the screen. Tap `+` to add a profile, or long-press any chip to rename or delete it.

### Tracker
The home screen shows:
- A smart insight derived from historical data ("Aria usually wakes around 02:58")
- The most recent wakeup entry
- Tonight's full log (entries since the night window began)

### Log a Wakeup
Tap **New entry** to open the log sheet:
- Timestamp is pre-filled to now; tap *Edit* to adjust it
- Pick one or more reasons from: Hunger, Wet / Diaper, Too Hot / Cold, Tummy, Comfort, Fever, Disrupted Sleep
- Add a free-text note
- Save or discard

### Analysis
Per-profile statistics:
- Total wakeups and average per night
- Usual wake time
- Top reason
- 30-minute wakeup histogram (Swift Charts)
- Reason breakdown chart

### Meds
Track medication doses with a safety countdown:
- Ibuprofen (6-hour default interval), Panadol / Paracetamol (4-hour), or custom
- Adjustable interval per dose
- Live countdown to next safe dose; turns green when ready

## Tech stack

| Layer | Technology |
|---|---|
| UI | SwiftUI |
| Persistence | Custom `NightStore` backed by a single JSON file in Application Support |
| Charts | Swift Charts (iOS 16+) |
| Project generation | XcodeGen (`project.yml`) |
| Linting | SwiftLint (`.swiftlint.yml`) |
| Tests | XCTest unit tests (`NightTrackerTests/`) |

## Setup & run

**Prerequisites:** macOS with Xcode 15+, and Homebrew.

```bash
# One-time tool install
brew install xcodegen swiftlint

# Generate the Xcode project
xcodegen generate

# Open and run (Cmd+R)
open NightTracker.xcodeproj
```

**Command-line build & test:**

```bash
xcodebuild -scheme NightTracker \
  -destination 'platform=iOS Simulator,name=iPhone 16' \
  build

xcodebuild test -scheme NightTracker \
  -destination 'platform=iOS Simulator,name=iPhone 16'
```

## Notes

- `NightTracker.xcodeproj` is generated by XcodeGen and is not committed. Always run `xcodegen generate` after cloning or pulling.
- First launch seeds two demo profiles with sample wakeups so you can explore the UI immediately.
- The store resets automatically if an incompatible data format is detected on launch.
- Launch arguments used by tests (ignored in normal use): `-startTab N`, `-openLog YES`.

> Native **iOS** app — runs on iPhone and in the iOS Simulator. Not available for Android.
